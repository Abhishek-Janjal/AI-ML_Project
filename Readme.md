# Project Index

1. [**Hand Gesture Controller**](Hand_control_3D_Object/handtracking.py)  
   Control and interact with 3D objects using real-time hand tracking via webcam.

2. [**Object Detection (Facial Emotion)**](Object_Detection/Facial_Emotion_Dataset.ipynb)  
   Detect and classify human facial emotions using a machine learning model.

3. [**RAG (Gemini)**](RAG/RAG_using_Gemini.ipynb)  
   Retrieval-Augmented Generation pipeline using Google Gemini for document-based Q&A.

4. [**RAG (Open Source)**](RAG/RAG_using_OS.ipynb)  
   Open-source RAG implementation using local LLMs and vector databases.

5. [**Vectorless RAG**](RAG/Vectorless_RAG.ipynb)  
   RAG approach without vector databases using alternative retrieval strategies.

6. [**Video Generation**](Video_Generation/newsVideo.ipynb)  
   Generate automated news-style videos from text input.

7. [**Quantum Machine Learning (QSVM)**](QML/QSVM_Qiskit.ipynb)  
   Implementation of Quantum Support Vector Machine using Qiskit.

8. [**nanoGPT**](nanoGPT/train.py)  
   Lightweight GPT training framework for experimenting with transformer models.

9. [**EEG Machine Learning**](EEG_ML_classification/Paper_Music.ipynb)  
   Classification of EEG signals (e.g., music vs rest states).
